Main Navigation

Styled in #4E5166 bg colour, block display and 0.3em 0.3em 0em 16% padding.

Nav items use inline-block, 0.7em 0.7em padding, #FFFFFF bold text in 2.5em size. Transition is a 0.15 sec background-colour ease out into #7B80A0.

CSS selector:

.nav-main .nav-main-item

Sample code:

<nav class="nav-main"> <a href="#" class="nav-main-item">Home</a> <a href="#" class="nav-main-item">Orders</a> <a href="#" class="nav-main-item">About</a> <a href="#" class="nav-main-item">Contact</a> </nav>

Rendered element:

Home Orders About Contact
